You do not like the "cookie policy" tab? Just deactivate it. You can use one of these shortcodes anywhere in your wordpress installation instead:

  • [cc_revoke_settings_link_nsc_bar]: only for compliance type 'opt-in' or 'opt-out'. Deprecated, will be removed in future versions. Please use other shortcode.
  • [cc_show_cookie_banner_nsc_bar]: for all compliance types (recommended).
